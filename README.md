
# DCS Lab Network MonitoringSystem


## Project Setup
- First download & install python 
- Change the directory to Backend
```bash
cd Backend
- Create vetual enviorenment
```bash
python -m venv venv
```
or
```bash
python3 -m venv venv
```

- Activate vertual enviorenment
Windows
```bash
venv\Scripts\activate
```

Linux
```bash
source venv/bin/activate
```
- Install dependncies
```bash
pip install -r requirements.txt
```
## Now run python scripts from
```bash
python filename.py
```
